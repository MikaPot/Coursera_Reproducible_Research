Fill in all of the missing values using the mean for that 5-minute interval and create a new dataset.

newData <- data %>% group_by(interval) %>%
    mutate(steps=ifelse(is.na(steps), 
        mean(steps, na.rm=TRUE),steps)) %>%
    ungroup

The differences between weekdays and weekends

add a new factor variable indicating whether a given date is a weekday or weekend day.

newData <- newData %>%
    mutate(week=ifelse(weekdays(date)%in%c("Sunday","Saturday"),
            "weekend", "weekday")) %>%
    mutate(week=as.factor(week))

Make a panel plot containing a time series plot of the interval and the average number ov steps taken

newData %>% group_by(interval, week) %>%
    summarise(ave_steps = mean(steps)) %>%
    ggplot(aes(x=interval, y=ave_steps)) +
    geom_line(aes(color=week)) +
